## ğŸš€ Getting Started

### Prerequisites

- Node.js 18 or later
- pnpm 8 or later

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/rasengan-dev/nindo.git
   cd nindo
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Start the development server:
   ```bash
   pnpm dev
   ```

4. Open [http://localhost:5320](http://localhost:5320) in your browser.
